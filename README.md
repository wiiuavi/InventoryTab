# setup
Clone the repo, open chrome, go to settings, On startup, select the option "open specific pages" and remove any entires there alrdy. Then add your path to the index.html of the repo.

# usage
Upon startup, the inv newtab will open.  
Here, you can type / to quickly type in the search bar and click enter to search. (There isnt a search button, but if you can't type I don't think you will get that far).  
Here you can also click **E**, **Escape** or press the book in the bottom left to access the inv.  
At first it will be empty but you can click the recipe book to add bookmarks, slot 1 refers to the top left, slot (the last number) refers to the bottom right, incrementing left to right, top to bottom.  
You may also choose where to position the search bar and clock widget, and change colour theme more to suit your wallpaper.
_**To delete a bookmark, leave both fields empty and save to the slot you wish to clear.**_  
If you cloned the repo, you can go to extensions, enable dev mode, and load unpacked, select the repo. now your newtab will use this.

To change the wallpaper, you should go to the cloned repo, and replace background1.png with a wallpaper of choice. You may do the same with the armour pieces to "equip" armour (this wont affect the player preview).  
A good place to start with the icons is here: (https://github.com/InventivetalentDev/minecraft-assets)
Use NPC Studio and make your own pose! or head to NameMC, click on a skin of choice, and use the camera icon to take a picture of the skin from any angle!
